# Changelog

## [0.1.0] - 2026-06-25

### Added

- Initial release
- Floating action button with toggle panel
- Font size adjustment (off, +10%, +20%, +30%)
- Line height adjustment (off, 1.5x, 2x, 2.5x)
- Text alignment (off, left, center, right, justify)
- Reading aid with adjustable size
- Desaturate (grayscale) mode
- Dyslexia-friendly font toggle
- Big cursor toggle
- Stop animations toggle
- Persistent settings via localStorage
- Configurable labels, theme colors, and positioning
- SSR-safe with Next.js support
- Runtime CSS injection (no separate CSS import required)
